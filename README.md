# python-calculator-with-history
 A beginner-friendly command-line calculator built using Python.   It supports basic arithmetic operations and stores calculation history in a file.

 ## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division (with zero division handling)
- 📜 View calculation history
- 🧹 Clear history
- 💾 Saves history to a file (`history.txt`)
- 🧠 Accepts both formats:
  - `5+5`
  - `5 + 5`

---
## 🛠️ Technologies Used

- Python 3
- File Handling

---

💻 Usage
Enter calculation(+,-,*,/) or command (history, clear)
Examples:
5+5
10-3
6*2
8/2
Commands:
history → Show all previous calculations
clear → Clear history
exit → Exit program

📁 Example Output
Enter calculation: 5+5
Result: 10

Enter calculation: history
5+5=10

⚠️ Error Handling
Invalid input format handled
Division by zero restricted
Only supports: + - * /
